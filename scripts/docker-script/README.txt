Como usar el scrtipt:
- Crea una carpeta y dentro guarda este script y el fichero .env
  El repositorio se va a clonar dentro de esta carpeta

- Hay dos maneras de ejecutar el script:
	- por linia de comando: de esta manera le puedes especificar tu feature branch y el script hara checkout a ella
	- haciendo doble click en el script: hara checkout a develop automaticamente

- Cuando el script finalice, ejecutar "composer update" en la terminal de docker
